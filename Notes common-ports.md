# Common Network Ports Reference
* `ls` - List all files and directories in the current folder.

* `pwd` - Print working directory (shows where you are).
 
* `cd` - Change directory to move between folders.
 
* `cat` - Concatenate and display the raw text content of a file.
 
* `grep` - Search and filter for text patterns.

* `chmod +x` - Modify file parameters to make a script executable.
